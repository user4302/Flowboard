# Usage Guide

This guide covers how to use Flowboard effectively for project management.

## Getting Started

### Creating Your First Board
1. Launch Flowboard in your browser
2. Click "Create New Board" in the sidebar
3. Enter a board title (e.g., "Website Redesign")
4. Add an optional description
5. Click "Create Board"

### Understanding the Interface
- **Sidebar**: Navigate between boards and access quick actions
- **Header**: Search, view switching, and main controls
- **Main Area**: Current board view (Kanban, Timeline, Calendar, or Table)
- **Modals**: Detailed card editing and board management

## Kanban View

### Basic Operations
- **Add Lists**: Click "Add a list" at the end of the board
- **Add Cards**: Click "Add a card" at the bottom of any list
- **Edit Cards**: Double-click any card to open details
- **Move Cards**: Drag cards between lists to change status
- **Reorder Lists**: Drag list headers to rearrange columns

### Card Management
1. **Quick Add**: Click "+ Add a card" and press Enter
2. **Detailed Edit**: Double-click to open full card editor
3. **Card Details**:
   - Title and description
   - Labels for categorization
   - Member assignments
   - Due dates
   - Priority levels (1-100)
   - Checklists for subtasks

### List Management
- **Rename**: Click the list title and edit inline
- **Delete**: Use the menu (⋮) → Delete list
- **Reorder**: Drag the list header to a new position

## Timeline View

### Navigation
- **Zoom Levels**: Use buttons 1-5 or click zoom controls
- **Date Navigation**: Use arrow keys or calendar picker
- **Scroll**: Horizontal scrolling for different time periods

### Features
- **Swimlanes**: Cards grouped by parent relationships
- **Hidden Cards**: Indicators for cards outside current view
- **Drag & Drop**: Reposition cards by dragging on timeline
- **Collapsible Lanes**: Click to expand/collapse swimlanes

### Zoom Levels
1. **Day**: Hour-by-hour view
2. **Week**: Daily breakdown
3. **2 Weeks**: Bi-weekly overview
4. **Month**: Monthly calendar
5. **Year**: Quarterly overview

## Calendar View

### Month Navigation
- **Previous/Next**: Use arrow buttons
- **Today**: Jump to current month
- **Date Selection**: Click any date to see details

### Card Display
- **Due Dates**: Cards appear on their due dates
- **Multi-day**: Cards spanning multiple dates
- **Color Coding**: Based on labels and priority

## Table View

### Columns
- **Title**: Card name
- **Status**: Current list
- **Assignee**: Assigned members
- **Due Date**: Card deadline
- **Priority**: Numerical priority
- **Labels**: Category tags

### Features
- **Sorting**: Click column headers to sort
- **Filtering**: Use search and filter options
- **Column Visibility**: Toggle column display

## Search and Filtering

### Search Bar
- **Global Search**: Finds cards across all boards
- **Real-time**: Results update as you type
- **Scope**: Searches titles, descriptions, and labels

### Advanced Filters
- **Labels**: Filter by specific labels
- **Members**: Show cards assigned to specific people
- **Completion**: Include/exclude completed cards
- **Priority**: Filter by minimum priority level
- **Due Dates**: Filter by time-based criteria

## Board Management

### Board Settings
- **Rename**: Click board name in sidebar
- **Delete**: Use delete button with confirmation
- **Export**: Download board data as JSON
- **Import**: Upload previously exported data

### Sharing (Coming Soon)
- **Invite Members**: Add collaborators via email
- **Permissions**: Control access levels
- **Real-time Sync**: Collaborate simultaneously

## Keyboard Shortcuts

### Global Shortcuts
- `Escape`: Close modals and dropdowns
- `Ctrl/Cmd + K`: Focus search bar
- `Ctrl/Cmd + /`: Show keyboard shortcuts

### Timeline View
- `1-5`: Switch zoom levels (Day to Year)
- `←/→`: Navigate time periods
- `↑/↓`: Navigate swimlanes
- `Home/End`: Jump to start/end of timeline

### Kanban View
- `Enter`: Confirm input (add card/list)
- `Escape`: Cancel editing
- `Tab`: Navigate between elements

## Tips and Best Practices

### Board Organization
- **Logical Lists**: Create lists that match your workflow (To Do, In Progress, Done)
- **Consistent Labels**: Use a standard set of labels across projects
- **Clear Titles**: Make card titles descriptive but concise
- **Regular Updates**: Keep due dates and statuses current

### Card Management
- **Break Down Tasks**: Use checklists for complex cards
- **Set Priorities**: Use the 1-100 scale for importance
- **Assign Owners**: Every card should have a responsible person
- **Add Context**: Use descriptions to provide necessary information

### Workflow Optimization
- **Daily Review**: Check and update cards daily
- **Weekly Planning**: Use timeline view for long-term planning
- **Sprint Planning**: Use calendar view for milestone tracking
- **Progress Tracking**: Use table view for status reports

## Troubleshooting

### Common Issues
- **Cards Not Saving**: Check browser localStorage permissions
- **Drag & Drop Not Working**: Ensure no modal is open
- **Search Not Finding**: Check spelling and filters
- **Timeline View Empty**: Verify cards have due dates

### Performance Tips
- **Large Boards**: Use filters to reduce visible cards
- **Browser Memory**: Refresh if application becomes slow
- **Mobile Devices**: Use Wi-Fi for better performance

## Data Management

### Backup Strategy
- **Regular Exports**: Export boards weekly
- **Multiple Copies**: Save exports in different locations
- **Version Control**: Keep multiple export versions

### Import Guidelines
- **File Format**: Only import valid JSON files from Flowboard
- **Data Validation**: Review imported data for accuracy
- **Duplicate Prevention**: Check for existing boards before import

### Migration
- **From Other Tools**: Use CSV import (if available)
- **To New Device**: Export from old device, import to new one
- **Account Changes**: Export before changing accounts
