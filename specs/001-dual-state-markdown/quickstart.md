# Quickstart: Dual-State Markdown Field

## Overview
This feature converts the task description field in the modal into a dual-state component:
- **Focused**: Raw Markdown editing.
- **Blurred**: Rendered Markdown view.

## Usage
1. Click on the description field to edit Markdown.
2. Click outside the field to see the rendered Markdown.

## Implementation Details
- Uses `react-markdown` for rendering.
- Tailwind 4 styles apply Markdown formatting.
