# Quickstart: Mobile Bottom Navigation

## Goal
Implement a fixed bottom navigation bar for mobile viewports (< 768px).

## Core Patterns
1. **Layout Integration**: Wrap the main view content to accommodate the fixed bottom bar height.
2. **Component Structure**: `MobileBottomNav` component with tab buttons.
3. **Tailwind Breakpoints**: Use `md:hidden` to hide on desktop.
