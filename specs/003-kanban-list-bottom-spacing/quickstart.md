# Quickstart: Verifying Kanban Bottom Spacing

## Verification Steps

1. Open a board in **Kanban View**.
2. Identify a list with many cards, or add enough cards to trigger a vertical scrollbar.
3. Scroll to the bottom of the list.
4. Verify that the "+ Add card" button group has a clear gap (minimum 16px) before the list's bottom border.
5. Drag a card from another list and drop it into the bottom-most position of this list.
6. Verify the drop is successful and the card is correctly placed as the last item.

## Expected Visuals
The buttons should not be flush with the bottom edge. There should be a visible background color gap below them.
